Quem somos?

O Banco da Saúde é uma Entidade Solidária, que se está a servir de uma Plataforma Informática para cumprir de forma o seu objeto social e conseguir fazer chegar os medicamentos a quem, não tendo dinheiro para os adquirir, deles necessita.

Onde nos encontrar?
